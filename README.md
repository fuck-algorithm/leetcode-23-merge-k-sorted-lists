# 23. 合并 K 个升序链表 - 算法可视化

LeetCode 第 23 题「合并 K 个升序链表」的算法可视化演示。

## 在线演示

👉 [点击查看在线演示](https://fuck-algorithm.github.io/leetcode-23-merge-k-sorted-lists/)

## 功能特性

- 支持三种解法：顺序合并、分治合并、优先队列
- 支持四种编程语言代码展示：Java、Python、Go、JavaScript
- 分步骤动画演示算法执行过程
- 代码高亮显示当前执行行
- 支持自定义输入数据
- 支持播放控制和进度拖拽

## 本地开发

```bash
npm install
npm run dev
```
